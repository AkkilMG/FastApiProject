# FastApiProject

### Vars

- `DATABASE_URL`: MongoDB database link (`dbName`: `studentDB`, `collectionName`: `studentCol`)


### Installation

```sh
git clone https://github.com/HeimanPictures/FastApiProject
cd FastApiProject
pip3 install -r requirements.txt
uvicorn main:app --reload
```
